# ASP Frontend

Vite + React + Ant Design frontend for Agentic SOC Platform.

## Development

```powershell
pnpm install
pnpm dev
```

The dev server proxies backend API requests under `/api` and websocket requests under `/ws`.

## Checks

```powershell
pnpm exec eslint .
pnpm exec tsc -b
```
